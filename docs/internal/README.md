# Internal

This directory holds non-product metadata that should not be treated as part of the application surface.

- `assistant/` contains assistant-specific prompts, skills, and related metadata that were moved out of the app tree to keep the public repository shape product-first.
- `REBRAND_REFRAME_STATUS.md` is the canonical handoff for the current rebrand and reframing work.
- `DOC_MAP.md` is the canonical index of every `.md` file and both `docs/` folders, including what to read vs skip and token optimization guidance.
- `WORKLOG.md` is the running activity log — check and update at the start and end of each assistant session.
